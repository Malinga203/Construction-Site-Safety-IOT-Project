🦺 Smart Safety Vest with RFID Tracking
📌 Project Overview

The Smart Safety Vest with RFID Location Tracking is an IoT-based system designed to improve safety in construction sites.
The vest uses sensors and RFID technology to monitor worker location and environmental conditions in real time.

🎯 Objectives

✔ RFID-based worker identification
✔ Real-time worker location tracking
✔ Hazardous gas detection
✔ Temperature monitoring
✔ Fall detection using motion sensors
✔ Real-time alerts for supervisors

⚙️ System Components
Hardware,
✔ NodeMCU (ESP8266 / ESP32)
✔ RFID Tag
✔ RFID Reader (RC522)
✔ MQ-2 Gas Sensor
✔ MQ-135 Gas Sensor
✔ DHT22 Temperature Sensor
✔ MPU6050 Sensor
✔ LED and Buzzer

Software,
✔ Arduino IDE
✔ Firebase Realtime Database
✔ Web Monitoring Dashboard
✔ HTML / CSS / JavaScript

🚀 Key Features

✔ Real-time worker tracking
✔ Restricted zone alerts
✔ Gas exposure monitoring
✔ Temperature safety monitoring
✔ Fall detection system
✔ Cloud-based monitoring
