🦺 Smart Safety Vest with RFID Tracking | IoT Safety Monitoring

A smart IoT-based safety system designed to improve worker protection in construction sites.
This project uses sensors and RFID technology to monitor worker locations, environmental conditions, and safety risks in real time.

🛠 Technologies Used

NodeMCU (ESP8266 / ESP32)

Arduino IDE

Firebase Realtime Database

HTML / CSS / JavaScript

RFID Technology

⚙ Hardware Components

RFID Tag

RFID Reader (RC522)

MQ-2 Gas Sensor

MQ-135 Gas Sensor

DHT22 Temperature Sensor

MPU6050 Motion Sensor

LED Indicator

Buzzer

🎯 Objectives

RFID-based worker identification

Real-time worker location tracking

Hazardous gas detection

Temperature monitoring

Fall detection using motion sensors

Real-time alerts for supervisors

✨ Features

Real-time worker tracking

Restricted zone alerts

Gas exposure monitoring

Temperature safety monitoring

Fall detection system

Cloud-based monitoring dashboard
